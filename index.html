🎴 TCG Drop Bot

A self-hosted personal deal-hunter for trading cards.

Self-hosted Personal use Python · FastAPI · Discord SQLite Pokemon TCG · One Piece TCG
What this is: A Discord bot + web dashboard that monitors major TCG retailers (Best Buy, Walmart, Target, Pokemon Center) and marketplaces (eBay, TCGplayer, Shopify shops) for restocks and below-market deals. Runs 24/7 on a personal home server. Used by one person.

Why I'm requesting Collectr API access

Collectr aggregates pricing for sealed and graded TCG products with strong coverage of Pokemon (English + Japanese) and One Piece — including Pokemon Center exclusives, Japanese-set cards, and graded slabs that are underrepresented in TCGplayer-only data sources. As a co-primary pricing source alongside TCGplayer/JustTCG, Collectr would let me:

  • Cross-validate market prices — when sources agree, the bot fires high-confidence alerts; when they diverge, it flags listings for manual review instead of mis-alerting.
  • Cover Pokemon Center exclusives + Japanese sealed — these are the products with the thinnest TCGplayer-only data and the fastest sellouts at retail.
  • Track condition tiers — Collectr's condition coverage helps separate NM-only deals from LP/MP listings that are already discounted.

Usage profile

Volume
~1,000–3,000 price lookups per day across ~25,000 catalog products
Pattern
Mostly nightly batch refresh + ad-hoc lookups when the bot detects a candidate listing
Caching
SQLite price-snapshot cache; 12-hour TTL by default to minimize repeat calls
Concurrency
Single home-server instance, ≤1 request/second to any source
Coverage scope
Pokemon (English + Japanese) and One Piece TCG — sealed and singles

Architecture

  • Catalog: ~25,000 products imported from TCGCSV (TCGplayer mirror), filtered to ≥$1 modern / ≥$5 vintage
  • Source layer: ~11 polling adapters (mainly eBay) running every 60s
  • Matching: multi-tier resolver (UPC → TCGplayer ID → set+name+number+variant → fuzzy)
  • Pricing: source-aggregator pattern with mean/min/max strategies; Collectr would slot in alongside JustTCG/TCGplayer/eBay-sold
  • Alerts: Discord webhook with tier-routed notifications (must-buy / high / digest / quiet)

Source code & transparency

The bot is a personal project, not commercial. There is no public service, no resale of pricing data, no third-party access. All Collectr API calls would be made server-side from a single home IP, with the bot caching responses locally to minimize repeated calls.
